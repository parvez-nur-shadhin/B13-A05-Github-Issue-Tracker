#  What is the difference between map(), filter(), and forEach()?

Ans: 
    map(): When i pass a function inside map() it's called transformation function.In this function each elements satisfies the condition will complete the logic inside the function will be completed and will return to new array, Otherwise it will be filtered out.

    filter(): when i pass a function inside filter() it's called test function. It Iterate through every element of an array and tests the element. if the element pass the test then it will be a part of new array, otherwise it will be filtered out.

    forEach(): ForEach is a kind of loop where we pass an arrow function. it loop through every element of the array and complete the logic inside the arrow function
